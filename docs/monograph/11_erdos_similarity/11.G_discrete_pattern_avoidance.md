# Appendix 11.G: Discrete Density Survey for 4-Point Pattern Avoidance

## G.1 The Discrete Analog
The continuous Erdős Similarity Conjecture asks whether a set $E \subset \mathbb{R}$ of positive measure can avoid all similar copies of a finite set $F$. To build a robust computational heuristic, we study the discrete analog: determining the maximum density of a subset $A \subseteq \{1,\dots,N\}$ avoiding all affine copies of $F$. 

It is crucial to recognize that the discrete vanishing of density is **necessary but not sufficient** for the continuous ESC. A positive-measure continuous set may avoid all integer-rescaled copies of $F$ while still containing real-rescaled copies. This survey strictly provides heuristic computational evidence for the difficulty of pattern avoidance at finite scales.

## G.2 The 4-Term Arithmetic Progression
For $F = \{0,1,2,3\}$, affine copies are exactly 4-term arithmetic progressions (4-APs). By Szemerédi's theorem (1975), the maximum density $\delta(N) = r_4(N)/N \to 0$ as $N \to \infty$. The exact rate of this decay is an active area of research:
* **Lower Bound (Behrend-Rankin)**: There exist 4-AP-free sets of size $\approx N \cdot \exp(-c\sqrt{\log N})$.
* **Upper Bound (Gowers 2001)**: The density is bounded above by $\approx (\log\log N)^{-c}$.

### G.2.1 Exact Values via Integer Linear Programming
We utilize OR-Tools CP-SAT to execute an exact Integer Linear Programming (ILP) optimization.
For variables $`x_i \in \{0,1\}`$, we maximize $`\sum_{i=1}^N x_i`$ subject to the constraint:

$$
x_a + x_{a+d} + x_{a+2d} + x_{a+3d} \leq 3
$$

for all valid $a \geq 1$ and $d \geq 1$.

### G.2.2 Comparison with Asymptotic Bounds
Our explicit computations for $\delta(N)$ illustrate the finite-$N$ regime of Szemerédi's theorem. Note that the random baseline $2N^{-1/2}$ plotted for reference is a crude heuristic threshold where a random set of that density would expect $O(1)$ 4-APs; it is not a rigorous bound.

## G.3 Non-Arithmetic Configurations
The ESC requires avoidance of **all** finite sets, not just arithmetic progressions. We generalize our ILP solver to test a non-AP 4-point configuration, such as $P = \{0,1,3,4\}$.

**Constraints:**

$$
x_a + x_{a+d} + x_{a+3d} + x_{a+4d} \leq 3
$$

Comparing $`r_P(N)`$ with $`r_{\text{AP4}}(N)`$ allows us to observe whether discrete pattern-avoidance density fundamentally depends on the arithmetic structure of the specific configuration.

At finite $N$, the non-arithmetic pattern $P = \{0,1,3,4\}$ permits slightly denser packings than the 4-term AP. This suggests that pattern-avoidance density depends on the combinatorial rigidity of the configuration at finite scales, though both strictly vanish as $N \to \infty$ by multi-dimensional generalizations of Szemerédi's theorem.

## G.4 Relation to the Continuous Conjecture
While these discrete densities explicitly vanish as $N \to \infty$, the continuous Erdős Similarity Conjecture strictly requires control over *all* real scalings. The discrete survey provides heuristic intuition for the combinatorial difficulty of pattern avoidance, but it operates purely as a heuristic mathematical model. It does not formally resolve the continuous analytical problem, which remains widely open.
